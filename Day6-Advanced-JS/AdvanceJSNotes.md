***FlightBooking
map() → Converts strings into objects.
split() → Separates data parts.
Number() → Converts price into number.
isNaN() → Handles invalid prices.
filter() → Keeps required flights.
sort() → Sorts by ascending price.
JSON.stringify() → Converts output into JSON.

***AutomatedPayroll
Callback → Calculates tax percentage.
map() → Processes employee payroll.
Tax = salary × tax rate.
Net Salary = salary − tax.
Ternary Operator → Sets Premium/Standard status.
Template Literal → Displays payroll summary.
setTimeout() → Delays final output.
