Playwright → Used for browser automation and testing.
Installation → Install Playwright using npm commands.
Project Setup → Create a Playwright project structure.
Folder Structure → Organizes tests and configuration files.
test() → Defines a test case.
page → Represents the browser page/tab.
goto() → Opens a website URL.
click() → Clicks on web elements.
fill() → Types text into input fields.
locator() → Finds elements on a webpage.
Basic Locators → Use id, class, text, or XPath to locate elements.
expect() → Validates expected results.
Running Tests → Execute tests using terminal commands.
Headed Mode → Runs browser in visible mode.
Auto-wait → Automatically waits for elements/actions.
Screenshot → Captures webpage images during tests.
Login Test → Automates login functionality testing.
HTML Report → Shows test execution results visually.
beforeEach() → Runs before every test case.
codegen → Generates Playwright code automatically.
