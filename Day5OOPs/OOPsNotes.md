Object – A collection of properties and methods used to store related data and behavior.
Class – A blueprint for creating objects with common properties and methods.
Constructor – A special method used to initialize object values when an object is created.
Object Literal – A simple way to create objects using {} syntax.
Instance – A real object created from a class.
Encapsulation – Wrapping data and methods together inside a class or object.
Inheritance – A feature where one class acquires properties and methods of another class.
Polymorphism – The ability of methods to behave differently in different classes.
Abstraction – Hiding internal details and showing only essential features.
